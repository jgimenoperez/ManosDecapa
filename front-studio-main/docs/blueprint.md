# **App Name**: Manos Decapa Landing

## Core Features:

- Informational Sections: Display information about services, target audience, process, reasons to choose the service, gallery, and pricing.
- Contact Form: Implement a contact form to gather customer inquiries, including file upload functionality for image attachments.
- Gallery Display: Create an image gallery with a before/after effect on hover and lightbox functionality for detailed viewing.
- Map Embedding: Embed a Google Maps iframe for the physical location of the business.

## Style Guidelines:

- Primary color: Wood brown (#8B4513) to convey natural and earthy tones.
- Secondary color: Chocolate (#D2691E) complements the primary, adding warmth.
- Accent color: Warm orange (#F4A460) to highlight calls to action and important elements.
- Background color: Cream (#FFF8DC) for a soft and inviting backdrop.
- Text color: Dark brown (#2C1810) for readability and a classic feel.
- Headings font: 'Poppins' sans-serif font, bold.
- Body font: 'PT Sans' sans-serif font, regular.
- Use outline-style icons for services and features, ensuring consistency across the site.
- Maintain a clean and spacious layout with generous spacing to improve readability.
- Incorporate subtle fade-in animations on scroll to enhance user experience without overwhelming.
